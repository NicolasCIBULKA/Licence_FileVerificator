
TRUE MIME APP


Ce projet est une application java permettant de déterminer si l'extension d'un fichier correspond bien a son type MIME effectif. L'application ne couvre pas tous les types MIME mais seulement une partie que vous pouvez retrouver dans le dossier MIMEbdd.


Prerequis

Vous devez disposer d'un environnement de developpement sur PC (sous Linux): JDK1.9  (ou superieur):compilateur java (javac) et JVM (java) / Eclipse. Aucune autre installation n est requise.


Demarrage

L application propose deux interfaces d utilisation: une interface graphique et une interface en ligne de commande.
Vous pouvez utiliser l'application via le terminal en utilisant la ligne de commande "java -cp bin cli.CommandLine" en vous placant dans le dossier Projet. L interface graphique est egalement disponible via le terminal en utilisant l'option "-g".

Vous pouvez tester l application en utilisant la ligne de commande "java -cp bin cli.CommandLine -d testingcli". Ce test vous donne un apperçu de l utilisation possible de cette application.


Conseils d utilisation

Via le terminal, en utilisant la commande "java -cp bin cli.CommandLine -h" (toujours en se placant dans le dossier Projet), les differentes options vous seront listees. Notez que l option -f vous donne des informations supplementaires, il est conseille de l utiliser lorsqu un fichier n est pas coherent pour en savoir plus.


Usage

Cette application est destinee a un usage ludique afin de mieux comprendre comment son constitues les fichiers en informatique.


Fabrique avec

Ce projet a ete realise avec l IDE Eclipse et Gedit
